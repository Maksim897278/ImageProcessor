# ImageProcessor

Десктопное приложение для загрузки и обработки изображений. Разработано в рамках лабораторной работы (**вариант 25**) на Python с использованием **PyQt5** и **OpenCV**.

## Возможности

### Базовый функционал

- Загрузка изображений из файла (PNG, JPG, JPEG).
- Захват кадра с веб-камеры.
- Просмотр отдельных цветовых каналов: красный, зелёный, синий (меню **Color Channels**).
- Сброс к исходному изображению (меню **Edit → Reset to Original**).

### Индивидуальный функционал (вариант 25)

| № | Операция | Меню | Описание |
|---|----------|------|----------|
| 1 | Изменение размера | **Edit → 1. Resize Image** | Задание новой ширины и высоты (1–4096 px) |
| 7 | Увеличение яркости | **Edit → 7. Increase Brightness** | Повышение яркости в диапазоне 0–100 (HSV) |
| 14 | Зелёная линия | **Edit → 14. Draw Green Line** | Отрисовка линии по координатам начала и конца с настраиваемой толщиной |

## Требования

- Python 3.6
- Веб-камера — только для функции захвата снимка

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <ССЫЛКА_НА_ВАШ_РЕПОЗИТОРИЙ>
cd <ИМЯ_ПАПКИ_ПРОЕКТА>
```

### 2. Виртуальное окружение

**Windows (PowerShell / CMD):**

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

Для работы приложения используются: `numpy`, `opencv-python`, `PyQt5`.

### 4. Запуск

```bash
python main.py
```

## Использование

1. Нажмите **Load Image (File)** или **Capture from Webcam**, чтобы загрузить изображение.
2. Применяйте операции через меню **Color Channels** и **Edit**.
3. Чтобы вернуться к исходному кадру, выберите **Edit → Reset to Original**.

## Стек технологий

- **PyQt5** — графический интерфейс
- **OpenCV (cv2)** — чтение, преобразование и рисование на изображениях
- **NumPy** — работа с массивами пикселей

## Возможные проблемы

| Проблема                                | Решение                                                                                                                              |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| Веб-камера недоступна        | Проверьте подключение камеры и закройте другие приложения, использующие её |
| Ошибка при загрузке файла | Убедитесь, что файл в формате PNG/JPG/JPEG и не повреждён                                               |
| Ошибка импорта PyQt5 / cv2         | Активируйте venv и переустановите зависимости: `pip install -r requirements.txt`                       |
